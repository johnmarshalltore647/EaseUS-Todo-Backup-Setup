Open the application and click the "Restore" button. Select the backup file from your list, choose the files or folders you need, and click "Restore" to move them back to your computer.

**Can I stop a backup while it runs?**
Yes. You can click the "Pause" or "Stop" button in the progress window to halt any active task.

**Is my backup password protected?**
